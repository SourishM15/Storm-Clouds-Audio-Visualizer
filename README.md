# Storm-Clouds-Audio-Visualizer
Storm Clouds Audio Visualizer is a real-time audio-reactive storm simulation built in JavaScript with p5.js. Live microphone input drives rain, wind, clouds, lightning, and delayed thunder using FFT-based bass, mid, and treble analysis for an immersive audiovisual experience.

Test it out here!: https://editor.p5js.org/S04rByT3/sketches/FMgceCJTz
